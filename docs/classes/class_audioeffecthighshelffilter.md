Adds a high-shelf filter to an audio bus.

classref-introduction-group

## Description

A "high-shelf" filter controls the gain of all frequencies above `AudioEffectFilter.cutoff_hz<class_AudioEffectFilter_property_cutoff_hz>`.

This filter can be used to increase or decrease clarity of a sound.

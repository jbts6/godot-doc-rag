Generic 2D position hint for editing.

classref-introduction-group

## Description

Generic 2D position hint for editing. It's just like a plain `Node2D<class_Node2D>`, but it displays as a cross in the 2D editor at all times. You can set the cross' visual size by using the gizmo in the 2D editor while the node is selected.

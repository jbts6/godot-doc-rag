Adds a low-shelf filter to an audio bus.

classref-introduction-group

## Description

A "low-shelf" filter controls the gain of all frequencies below `AudioEffectFilter.cutoff_hz<class_AudioEffectFilter_property_cutoff_hz>`.

This filter can be used to adjust the "strength" of a sound, by increasing or decreasing its low-end.

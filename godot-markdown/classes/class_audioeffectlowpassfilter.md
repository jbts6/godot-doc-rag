Adds a low-pass filter to an audio bus.

classref-introduction-group

## Description

A "low-pass" filter attenuates frequencies higher than `AudioEffectFilter.cutoff_hz<class_AudioEffectFilter_property_cutoff_hz>` and allows lower frequencies to pass unchanged.

This filter can be used to muffle sounds.

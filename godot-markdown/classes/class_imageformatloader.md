# ImageFormatLoader

**Inherits:** `RefCounted<class_RefCounted>` **\<** `Object<class_Object>`

**Inherited By:** `ImageFormatLoaderExtension<class_ImageFormatLoaderExtension>`

Base class to add support for specific image formats.

classref-introduction-group

## Description

The engine supports multiple image formats out of the box (PNG, SVG, JPEG, WebP to name a few), but you can choose to implement support for additional image formats by extending `ImageFormatLoaderExtension<class_ImageFormatLoaderExtension>`.

# HScrollBar

**Inherits:** `ScrollBar<class_ScrollBar>` **\<** `Range<class_Range>` **\<** `Control<class_Control>` **\<** `CanvasItem<class_CanvasItem>` **\<** `Node<class_Node>` **\<** `Object<class_Object>`

A horizontal scrollbar that goes from left (min) to right (max).

classref-introduction-group

## Description

A horizontal scrollbar, typically used to navigate through content that extends beyond the visible width of a control. It is a `Range<class_Range>`-based control and goes from left (min) to right (max).
